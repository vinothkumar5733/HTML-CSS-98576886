# Resume Website

A lightweight static resume website built with plain HTML and CSS.

## Repository structure

- `index.html` — the main resume page with a two-column layout, profile image, contact details, technical skills, education, projects, and achievements.
- `sample.html` — an example landing page with navigation links, about section, project list, and contact information.
- `images/` — contains the profile photo used on `index.html`.

## Features

- Static resume layout with CSS grid
- Personal profile and contact section
- Education, projects, achievements, and skills sections
- Simple sample page with navigation and contact details

## Usage

1. Open `index.html` or `sample.html` in a web browser.
2. Customize the HTML text, links, and profile image.
3. Replace the image in `images/` with your own photo.

## Recommended improvements

- Use a separate CSS file for maintainability.
- Add a descriptive page title and meta description.
- Fix placeholder links in `sample.html` and point them to real sections or URLs.
- Improve accessibility by using semantic HTML and better contrast.

## Notes

This repo is designed for a simple static resume site. No build tools are required.
